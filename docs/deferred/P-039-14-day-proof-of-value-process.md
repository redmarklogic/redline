# 14-day Proof-of-Value (POV) process for the 10-partner phase

## Why deferred

Founder reviewed GTM research (Koi sales podcast, Amit Yampolsky) and chose to defer to
Sprint 3. Phase-1 / current-sprint priorities take precedence; no partner is live yet, so
formalising the trial process now is premature. Source research:
[`../product/gtm/20260608-koi-sales-podcast-pov-research.md`](../product/gtm/20260608-koi-sales-podcast-pov-research.md).

Decision on the table at retrieval:
- **Adopt** — formalise a 14-day POV: one use-case, value-based acceptance criterion written
  before day 1, day-7 summation checkpoint, day-14 keep/kill decision.
- **Adopt-lite** — run it informally with the first partner, no written PRD.
- **Drop** — no structured POV.
